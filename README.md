# CatchCoinsGame-with-Lanterna
Simple catch coins game made with the Lanterna library in Java.  
# Implementation
Initial implementation was made in Lab classes of LDTS@FEUP following the instructions present here: https://github.com/ruimaranhao/FEUP-LEIC-LDTS-Labs/blob/main/Lab02-03.md.  
Further improvements were made by me on top of the initial implementation.  
# How to run
1) Clone this repository locally.
2) In its directory, run
```
./gradlew run
```
3) Play!
